This test asks whether the following execution is architecturally Allowed:

\begin{itemize}
  \item The Store instruction on P0, i.e. STR W0,[X1], generates an Explicit Memory Write Effect of Location x with value 2;
  \item The first Load instruction in program order on P1, i.e. LDR W4,[X1], generates an Explicit Memory Read Effect of Location x;
  \item The second Load instruction in program order on P1, i.e. LDR W5,[X2], generates an Explicit Memory Read Effect of Location y;
  \item The Store instruction on P2, i.e. STR W0,[X2], generates an Explicit Memory Write Effect of Location y with value 1;
  \item The Load instruction on P2, i.e. LDR W4,[X3], generates an Explicit Memory Read Effect of Location z;
  \item The first Store instruction in program order on P3, i.e. STR W0,[X3], generates an Explicit Memory Write Effect of Location z with value 1;
  \item The second Store instruction in program order on P3, i.e. STR W5,[X1], generates an Explicit Memory Write Effect of Location x with value 1;
  \item The value of x is 2 in the end, which means that the Explicit Memory Write Effect of Location x on P3 generated by STR W5,[X1] is Coherence-before the Explicit Memory Write Effect on P0;
  \item The Register X4 on P1 holds the value 2 in the end, which means that the Explicit Memory Read Effect of Location x on P1 generated by LDR W4,[X1] Reads-from the Explicit Memory Write Effect on P0;
  \item The Register X5 on P1 holds the value 0 in the end, which means that the Explicit Memory Read Effect of Location y on P1 generated by LDR W5,[X2] Reads-from the initial state, and therefore is Coherence-before the Explicit Memory Write Effect on P2;
  \item The Register X4 on P2 holds the value 0 in the end, which means that the Explicit Memory Read Effect on P2 Reads-from the initial state, and therefore is Coherence-before the Explicit Memory Write Effect of Location z on P3 generated by STR W0,[X3].
\end{itemize}
