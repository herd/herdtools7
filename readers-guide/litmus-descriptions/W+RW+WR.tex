This test asks whether the following execution is architecturally Allowed:

\begin{itemize}
  \item The Store instruction on P0, i.e. STR W0,[X1], generates an Explicit Memory Write Effect of Location x with value 1;
  \item The Load instruction on P1, i.e. LDR W4,[X1], generates an Explicit Memory Read Effect of Location x;
  \item The Store instruction on P1, i.e. STR W2,[X3], generates an Explicit Memory Write Effect of Location y with value 1;
  \item The Store instruction on P2, i.e. STR W0,[X3], generates an Explicit Memory Write Effect of Location y with value 2;
  \item The Load instruction on P2, i.e. LDR W4,[X1], generates an Explicit Memory Read Effect of Location x;
  \item The value of y is 2 in the end, which means that the Explicit Memory Write Effect on P1 is Coherence-before the Explicit Memory Write Effect on P2;
  \item The Register X4 on P1 holds the value 1 in the end, which means that the Explicit Memory Read Effect on P1 Reads-from the Explicit Memory Write Effect on P0;
  \item The Register X4 on P2 holds the value 0 in the end, which means that the Explicit Memory Read Effect on P2 Reads-from the initial state, and therefore is Coherence-before the Explicit Memory Write Effect on P0.
\end{itemize}
