This test asks whether the following execution is architecturally Allowed:

\begin{itemize}
  \item The Store instruction on P0, i.e. STR W0,[X1], generates an Explicit Memory Write Effect of Location x with value 1;
  \item The first Load instruction in program order on P1, i.e. LDR W1,[X0], generates an Explicit Memory Read Effect of Location x;
  \item The second Load instruction in program order on P1, i.e. LDR W3,[X2], generates an Explicit Memory Read Effect of Location y;
  \item The Store instruction on P2, i.e. STR W0,[X1], generates an Explicit Memory Write Effect of Location y with value 1;
  \item The first Load instruction in program order on P3, i.e. LDR W1,[X0], generates an Explicit Memory Read Effect of Location y;
  \item The second Load instruction in program order on P3, i.e. LDR W3,[X2], generates an Explicit Memory Read Effect of Location x;
  \item The Register X1 on P1 holds the value 1 in the end, which means that the Explicit Memory Read Effect of Location x on P1 generated by LDR W1,[X0] Reads-from the Explicit Memory Write Effect on P0;
  \item The Register X3 on P1 holds the value 0 in the end, which means that the Explicit Memory Read Effect of Location y on P1 generated by LDR W3,[X2] Reads-from the initial state, and therefore is Coherence-before the Explicit Memory Write Effect on P2;
  \item The Register X1 on P3 holds the value 1 in the end, which means that the Explicit Memory Read Effect of Location y on P3 generated by LDR W1,[X0] Reads-from the Explicit Memory Write Effect on P2;
  \item The Register X3 on P3 holds the value 0 in the end, which means that the Explicit Memory Read Effect of Location x on P3 generated by LDR W3,[X2] Reads-from the initial state, and therefore is Coherence-before the Explicit Memory Write Effect on P0.
\end{itemize}
