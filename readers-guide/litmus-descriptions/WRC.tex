This test asks whether the following execution is architecturally Allowed:

\begin{itemize}
  \item The Store instruction on P0, i.e. STR W1,[X0], generates an Explicit Memory Write Effect of Location x with value 1;
  \item The Load instruction on P1, i.e. LDR W1,[X0], generates an Explicit Memory Read Effect of Location x;
  \item The Store instruction on P1, i.e. STR W2,[X3], generates an Explicit Memory Write Effect of Location y with value 1;
  \item The first Load instruction in program order on P2, i.e. LDR W1,[X3], generates an Explicit Memory Read Effect of Location y;
  \item The second Load instruction in program order on P2, i.e. LDR W4,[X0], generates an Explicit Memory Read Effect of Location x;
  \item The Register X1 on P1 holds the value 1 in the end, which means that the Explicit Memory Read Effect on P1 Reads-from the Explicit Memory Write Effect on P0;
  \item The Register X1 on P2 holds the value 1 in the end, which means that the Explicit Memory Read Effect of Location y on P2 generated by LDR W1,[X3] Reads-from the Explicit Memory Write Effect on P1;
  \item The Register X4 on P2 holds the value 0 in the end, which means that the Explicit Memory Read Effect of Location x on P2 generated by LDR W4,[X0] Reads-from the initial state, and therefore is Coherence-before the Explicit Memory Write Effect on P0.
\end{itemize}
