# Building the HTML version of the ASL Reference

## Prerequisite

Install hevea either from sources <https://github.com/maranget/hevea>, or
with opam (version 2.38 or higher).
```
% opam update
% opam install hevea
% hevea -version
hevea 2.38
...
```

## Proceed

Run `sh ./zyva.sh` in this directory. The HTML Reference is in
subdirectory `HTML/`.
